Important points to remember

for loop is best when the number of repetitions is already known.
The counter variable keeps track of the loop progress.
The loop stops when the condition becomes false.
If the condition is always true, the loop becomes infinite.
while loop is better when the number of repetitions is not fixed in advance.
Difference between for and while

for loop: start, condition, and increment are written together.
while loop: the variable usually starts outside, and the increment is written inside the block.
for loop is cleaner when counting in a planned way.
while loop is useful when the stop point depends on a condition.
Final checkpoint answer

How is for different from while?

for loop is used when I know how many times the code should run, and its start, condition, and increment are written in one line. while loop is used when the number of repetitions is not fixed in advance, and the loop continues as long as the condition stays true.
