1) Very Short Revision
What functions are
A function is a reusable block of code that performs a task.
Function parameters
Parameters are inputs written in the function definition.
Return values
return sends a value back from the function.
Arrow functions
Arrow functions are a shorter way to write functions.
Scope basics
Scope decides where a variable can be used.
2) How These Topics Connect
A function works like this:
input → process → output
Parameters bring input into the function
The function body does the work
Return sends the result out
Scope controls which variables can be used
Arrow functions write the same idea in a shorter way
3) Core Notes to Remember
A function may take inputs.
A function may return a result.
If a function does not use return, the result is undefined.
Variables inside a function are usually local.
Arrow functions and normal functions do the same job.
4) Notebook Version

5) Practice Tasks
Task 1 — Add two numbers
Create a function that takes two numbers and returns their sum.
Example: add(4, 6) → 10
Task 2 — Check even or odd
Create a function that takes a number and returns "Even" or "Odd".
Example: checkNumber(7) → "Odd"
Task 3 — Final product price
Create a function that takes original price and discount, then returns the final price.
Example: finalPrice(1000, 150) → 850
Task 4 — Minutes to seconds
Create a function that takes minutes and returns seconds.
Example: toSeconds(5) → 300
Task 5 — Pass or Fail
Create a function that takes marks and returns "Pass" if marks are 40 or above, otherwise "Fail".
Example: result(55) → "Pass"
