What I Learned
Today I learned how JavaScript uses an if statement to make decisions.
An if statement runs code only when the condition inside it is true.

Example
let age = 18;

if (age >= 18) {
  console.log("You are an adult");
}
How it works
age >= 18 is the condition
If the condition is true, the code inside {} runs
If the condition is false, the code is skipped
Task
Show a message based on age using if.
Checkpoint
What makes code enter an if block?
Answer
The code enters an if block when the condition returns true.
Conclusion
The if statement is used to control program flow and run code only when a condition is satisfied.
